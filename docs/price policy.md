# 업비트의 기본 원칙

## 1. 업비트의 기본 원칙 (KRW 마켓)

### 가격(price)

원화 마켓은 정수 원 단위
반올림 없음, 소수점 가격 자체가 불가

### 수량(volume)

코인 수량은 소수점 허용
보통 최대 8자리

핵심: 항상 내림(truncate)
반올림 절대 안 함


## 수수료

### 거래 수수료 (매수·매도)

#### KRW 마켓

수수료율: 0.05%

매수·매도 동일

메이커 / 테이커 구분 없음

예시

1,000,000원 매수
→ 수수료 500원
→ 실제 체결 금액 기준으로 차감

주의

수수료는 KRW로 차감

코인 수량이 깎이지 않는다

#### BTC 마켓

수수료율: 0.25%

매수·매도 동일

예시

0.1 BTC 매수
→ 수수료 0.00025 BTC 상당

KRW 마켓보다 비싸기 때문에
웬만하면 직접 BTC 마켓을 쓰지 않는 이유다.

### 수수료가 빠지는 타이밍

#### 매수

KRW 잔고에서 차감

계산식
사용 금액 + 수수료 ≤ 보유 KRW

매도

매도 금액에서 차감

계산식
체결 금액 − 수수료 = 실수령 KRW

### 출금 수수료 (중요)

출금은 고정 수수료다.
거래 수수료랑 완전히 다르다.

예시

BTC 출금: 0.0008 BTC

ETH 출금: 0.01 ETH

특징

금액이 커도, 작아도 동일

네트워크 상황과 무관

업비트가 미리 정해둔 값
